Team Task Manager - Full Stack Web Application

Project Overview:
Team Task Manager is a full-stack web application that allows users to create accounts, log in, manage projects, assign tasks, and track progress.

Tech Stack:
Frontend:
- React.js
- Vite
- Axios
- React Router DOM

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs

Deployment:
- Frontend: Railway
- Backend: Railway

Features:
- User Registration
- User Login Authentication
- Role-based access (Admin / Member)
- Project creation
- Task management
- Dashboard view
- MongoDB database integration
- Live deployment
